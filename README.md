# proxy-server

## About

Initialization scripts for setting up a Linux machine as a DIY proxy server.

## Install

```
$ git clone https://github.com/jeffrade/proxy-server.git
$ cd proxy-server
$ bash install.sh
```

## What does this do?

 - Installs fail2ban
 - Installs nginx
 - Installs pi-hole
 - Installs AWS Route53 DNS updater
 - Installs AWS S3 Glacier uploader
